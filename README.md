## Introduce

여러 언론사에 제보하는게 불편하신가요?  
빠르게 내가 제보한 언론이 퍼졌으면 좋겠나요?  
제보를 퍼올때 너무 오랜시간동안 허가를 기다려야 하나요?  
낮은 기사화로 제보하는게 지치셨나요?

그런 당신을 위한 플랫폼 **Breaking**
